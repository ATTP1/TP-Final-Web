Animation utilisé (GSAP) pour:

    Titres: 

        Effet utilisé: 
            -transtion
            -stagger
            -delay
            -opacity

    Boîtes: 

        Effet utilisé: 
            -transtion
            -stagger
            -delay        
            -ease(bounce)

    Loading: 

        Effet utilisé: 
            -transtion
            -stagger
            -delay        
            -ease(bounce)
            -hidden
            -from() /to()
            -on complete()       

    ref:
        https://gsap.com/
        https://gsap.com/cheatsheet/
